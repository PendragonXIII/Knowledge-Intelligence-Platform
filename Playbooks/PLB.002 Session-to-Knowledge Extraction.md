## Purpose

Convert valuable AI sessions into reusable knowledge objects.

## Process

1. Review session outcomes.
2. Load relevant repository evidence.
3. Identify durable insights.
4. Classify findings as:
   - Learning
   - Constraint
   - Opportunity
   - Architecture Decision
5. Perform Knowledge Object Assessment.
6. Create knowledge objects.
7. Create relationships.
8. Review graph consistency.

## Success Criteria

Knowledge generated during sessions becomes:

- reusable
- discoverable
- linkable
- available for future AI context assembly
