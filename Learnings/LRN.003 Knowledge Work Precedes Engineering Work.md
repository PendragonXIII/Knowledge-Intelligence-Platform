# LRN.003 Knowledge Work Precedes Engineering Work

## Observation

Strategic decisions and capability prioritization should occur before implementation planning begins.

## Learning

Knowledge work and engineering work represent distinct phases.

The Knowledge Assistant determines what should be built.

The Engineering Assistant determines how it should be built.

## Outcome

Engineering implementation should consume structured implementation intent rather than derive strategic direction during implementation sessions.
