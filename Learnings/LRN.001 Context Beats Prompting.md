# LRN.001 Context Beats Prompting

## Observation

Prompt quality improvements produce diminishing returns.

Context quality improvements scale across all future interactions.

## Learning

Investing in reusable context creates greater long-term value than optimizing individual prompts.

## Implications

Prioritize:

- knowledge layer
- constraints
- playbooks
- capability relationships

before investing heavily in prompt engineering.
