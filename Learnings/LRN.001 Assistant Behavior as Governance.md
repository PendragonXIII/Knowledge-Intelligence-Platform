# LRN.001 Assistant Behavior as Governance

## Observation

Assistant behavior can be modeled as governed knowledge rather than embedded GPT instructions.

## Learning

Reasoning behavior, communication style, visualization preferences, context loading behavior and session behavior are governance assets.

## Outcome

Introduced:

- KTS.009 Assistant Profile Standard
- ASP.001 Knowledge Assistant Profile
- ASP.002 Engineering Assistant Profile
- ASP.003 Garden Assistant Profile
