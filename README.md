# Knowledge Intelligence Platform

## Purpose

The Knowledge Intelligence Platform provides a governed, AI-consumable knowledge repository for engineering intelligence, knowledge governance and context assembly.

The repository contains structured Knowledge Objects rather than traditional documentation.

## Knowledge Object Types

### Capabilities

Describe current and future system capabilities.

Prefix:

```text
EID
```

### Opportunities

Describe potential improvements, initiatives and future investments.

Prefix:

```text
OP
```

### Constraints

Define governance rules and operating principles.

Prefix:

```text
CNS
```

### Learnings

Capture durable insights and lessons learned.

Prefix:

```text
LRN
```

### Architectural Decision Records

Document significant architectural decisions.

Prefix:

```text
ADR
```

### Playbooks

Define repeatable processes and workflows.

Prefix:

```text
PLB
```

### Roadmaps

Define phased implementation plans.

Prefix:

```text
RMP
```

### Knowledge Template Standards

Define approved schemas and governance requirements.

Prefix:

```text
KTS
```

Knowledge Objects are governed through:

* Knowledge Template Standards (KTS)
* Constraints (CNS)
* Playbooks (PLB)
