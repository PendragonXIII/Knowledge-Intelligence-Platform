## Standard

[[KTS.001 Capability Standard]]

## Purpose

Compare extracted knowledge candidates against the Knowledge Repository to identify existing knowledge, missing knowledge, duplicate knowledge, conflicting knowledge and recommended repository changes.

## Current State

Operational

## North Star

Provide the ability for AI agents to continuously evaluate extracted knowledge against the Knowledge Repository and generate actionable governance recommendations.

## Agentic Contribution

AI agents can:

- compare extracted knowledge
- identify existing knowledge
- identify missing knowledge
- identify duplicate knowledge
- identify conflicting knowledge
- identify repository gaps
- recommend repository updates
- recommend knowledge creation
- recommend knowledge maintenance
- support governance reviews

## Outcomes

### O1 – Existing Knowledge Detection

Establish the ability to:

- identify existing knowledge objects
- identify existing capabilities
- identify existing constraints
- identify existing opportunities
- identify existing learnings

### O2 – Missing Knowledge Detection

Establish the ability to:

- identify missing knowledge objects
- identify missing capabilities
- identify missing constraints
- identify missing opportunities
- identify missing learnings

### O3 – Duplicate Knowledge Detection

Establish the ability to:

- identify duplicate knowledge
- identify overlapping knowledge
- identify redundant knowledge objects

### O4 – Conflict Detection

Establish the ability to:

- identify conflicting knowledge
- identify governance conflicts
- identify relationship conflicts
- identify consistency issues

### O5 – Repository Gap Analysis

Establish the ability to:

- identify repository gaps
- identify documentation gaps
- identify governance gaps
- identify capability gaps

### O6 – Change Recommendation Generation

Establish the ability to:

- recommend new objects
- recommend object updates
- recommend relationship updates
- recommend governance improvements

### O7 – Knowledge Comparison Services

Establish services that provide structured comparison of extracted knowledge and repository knowledge.

Services may support:

- object comparison
- relationship comparison
- gap analysis
- recommendation generation

## Scope

### In Scope

- Knowledge Comparison
- Gap Analysis
- Duplicate Detection
- Conflict Detection
- Change Recommendation
- Governance Recommendation

### Out of Scope

- Repository Modification
- Commit Creation
- Automatic Repository Updates
- Knowledge Object Creation
- Knowledge Object Maintenance

## Consumers

- Knowledge Assistant

- Future Garden AI Agents
